# 硬件加速指南

DJI Batch LUTer 支持多种硬件加速方案，以显著提升视频处理速度。

## 1. 支持的硬件加速技术

### 编码器 (Encoding)
- **NVIDIA NVENC**: 适用于 NVIDIA 显卡。
- **Intel QSV**: 适用于 Intel 核显。
- **AMD AMF**: 适用于 AMD 显卡。

### 解码器 (Decoding)
本工具目前在 GPU 模式下自动尝试启用硬件加速解码（如 CUDA, QSV, D3D11VA），以减轻 CPU 负担。

## 2. 常见问题排查

### 为什么我的显卡没有被识别？
1. 请确保已安装最新的显卡驱动。
2. 确保 FFmpeg 版本支持对应的编码器。可以通过命令行运行 `ffmpeg -encoders` 查看是否包含 `nvenc`, `qsv` 或 `amf`。

### 并行任务数设置多少合适？
- **GPU 加速**: 建议设为 1-2。显卡的硬件编码器并发能力有限，设置过多可能导致显存溢出或报错。
- **CPU 编码**: 建议设为 CPU 逻辑核心数。
